# Demo Seed Data

`sample_assessments.csv` is a curated demo cohort used to populate the Monitoring Dashboard during client demos.

To load it:

1. Run the Streamlit app.
2. Open the sidebar.
3. Enable **Show admin/demo tools**.
4. Click **Load Client Demo Data**.

The loader replaces the previous `demo_client_seed` batch before inserting the records, so repeated demo loads do not duplicate the seed cohort.
